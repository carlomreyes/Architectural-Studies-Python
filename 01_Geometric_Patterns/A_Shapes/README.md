# A_Shapes

This folder contains project folders + simple examples & starter notebook samples for exploring basic geometric shapes

Files:

- `example.py` — minimal script demonstrating drawing a circle and a square with `matplotlib`.
- `starter_notebook.ipynb` — a small starter Jupyter notebook for interactive exploration.

Usage:

1. To run the script (shows a matplotlib window):

```bash
python example.py
```

2. To use the notebook, open `starter_notebook.ipynb` in Jupyter or VS Code Notebook.

Dependencies:

- `numpy`
- `matplotlib`
