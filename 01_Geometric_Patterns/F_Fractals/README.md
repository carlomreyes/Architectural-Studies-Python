# F_Fractals

This folder contains examples and a starter notebook for exploring basic fractals (Sierpinski triangle).

Files:

- `example.py` — script that generates and plots the Sierpinski triangle using the chaos game.
- `starter_notebook.ipynb` — Jupyter notebook for interactive exploration.

Usage

Run the script to display a quick fractal plot:

```bash
python example.py
```

Open `starter_notebook.ipynb` in Jupyter or VS Code Notebook to run and modify parameters interactively.

Dependencies

- `numpy`
- `matplotlib`
